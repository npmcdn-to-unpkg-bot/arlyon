what is this for?

This site was built to fill in a few holes that were missing for me. I couldn't find a modern catalog tool for keep track of my stuff, and really like seeing my life in terms of raw data. How many shirts do I own? How long does the average pair of jeans last? There are some apps out there, but they all are either a) ugly or b) expensive. So I decided to write my own. To be honest, I'm not looking for mass distribution. This is my playground. If I help someone file an insurance claim great! Since this was designed for me, you can have it for free. So share it around, and as long as it's not too popular, I'll be able to host it myself. Here are some features I value in my apps:
